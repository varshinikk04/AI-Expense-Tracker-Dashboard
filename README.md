# 💰 AI-Powered Expense Tracker Dashboard

An interactive Expense Tracker Dashboard built using Python, Streamlit, Pandas, and Matplotlib. This application helps users record, manage, and analyze their daily expenses through an intuitive web interface and visual analytics dashboard.

---

## 🚀 Project Overview

Managing personal finances is an important aspect of financial planning. This project provides a simple and user-friendly solution for tracking expenses, visualizing spending patterns, and generating insights to support better financial decisions.

The dashboard allows users to add expenses, store records, view spending history, and analyze category-wise expenditure through interactive charts and summary metrics.

---

## ✨ Features

### 📌 Expense Management
- Add new expenses
- Store expense records permanently using file handling
- View all recorded expenses in a structured format

### 📊 Data Visualization
- Category-wise Bar Chart
- Expense Distribution Pie Chart
- Interactive Dashboard Layout

### 📈 Expense Analytics
- Total Expense Tracking
- Category Count Metrics
- Spending Pattern Analysis

### 🤖 AI-Powered Insights
- Identifies highest spending category
- Calculates category contribution percentage
- Provides personalized savings suggestions

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Expense-Tracker-Dashboard.git
```

### Navigate to Project Folder

```bash
cd AI-Expense-Tracker-Dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
